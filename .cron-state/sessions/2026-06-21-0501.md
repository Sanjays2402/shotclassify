### Code fields (1 new slot)
- `CodeFields.comment_density` — float in [0.0, 1.0] capturing
  the share of non-blank lines that open with a comment leader
  for the snippet's language. 40+ language tags catalogued
  across #, //, --, ;, %, <!-- families. Block-comment openers
  (/*, """, ''', =begin) count when at line start. PHP gets
  both // and # as leaders. Pure data formats (json/csv/tsv)
  always return 0.0; the text catchall defaults to #.

### Error extractor (2 new framework tags)
- `framework='php'` — Fatal error: Uncaught X: ... + Stack trace:
  + thrown in PATH on line N. Namespace-qualified exceptions
  preserved (Symfony\Component\HttpKernel\...). Placed BEFORE
  the JVM branch so PHP exception names don't tag as JVM. 13
  likely_cause hints covering TypeError, ArgumentCountError,
  DivisionByZeroError, class-not-found autoloader failure,
  generic Runtime/Logic/InvalidArgument exceptions, OutOfBounds,
  PDOException, and permission-denied filesystem errors.
- `framework='sql'` — engine-specific preludes for PostgreSQL
  (ERROR:/LINE N:), MySQL (ERROR NNNN (SQLSTATE):), SQLite
  (Error: with vocab anchor), MSSQL (Msg N, Level N, State N
  + next-line message). Sits in the else: branch BEFORE HTTP
  so SQL errors don't pre-empt real HTTP traces. Dialect
  identifier folded into the exception slot (MySQL 1064 (42000),
  PostgreSQL ERROR, SQLite Error, MSSQL Msg 207). 13 likely_cause
  hints across syntax errors, missing tables/columns, unique
  duplicates, foreign-key violations, deadlocks, lock timeouts,
  permission denied, data-too-long, NOT NULL violations.
